Configurar sincronización compartida
Sigue estos pasos en orden. Solo lo hace una persona — los demás solo pegan los datos.
Paso 1 — Ve a jsonbin.io → Sign Up gratis → entra a tu cuenta.
Paso 2 — Crear el Bin
Menú izquierdo → Bins → + Create Bin → borra el contenido y escribe {"init":true} → Create.
Luego abre el Bin creado → clic en Settings → activa Make bin public → guarda.
Paso 3 — Copiar el Bin ID
En la URL del navegador verás algo como jsonbin.io/b/64f3a1.../edit — ese número largo es el Bin ID.
Paso 4 — API Key
Menú izquierdo → API Keys → + Create API Key → nombre: turnos → crea y copia la key.
⚠ Comparte la API Key y el Bin ID solo por mensaje privado con tu equipo — no los publiques en ningún lado.
